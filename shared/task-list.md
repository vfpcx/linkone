# Task List for Superpowers Collaboration

## Team Lead
- Receive project requirements and break them down into tasks.
- Assign tasks to appropriate roles.
- Coordinate between roles to resolve conflicts and manage dependencies.
- Merge all code changes into the main project.
- Produce project summaries and delivery documentation.

## Architect Agent
- Select technologies and design the overall architecture.
- Define database schema, API specifications, and module divisions.
- Document architectural decisions, field definitions, and error codes.

## Backend Developer Agent
- Implement APIs, business logic, data CRUD operations, and permission checks.
- Handle password encryption, parameter validation, and exception management.
- Document APIs and synchronize with the frontend team.

## Frontend Developer Agent
- Develop page layouts, components, and interactive logic.
- Integrate with backend APIs, manage application state, and handle token storage.
- Ensure responsive design across different devices.

## Testing & Review Agent
- Write unit tests, integration tests, and boundary cases.
- Review code for compliance with coding standards and check for security vulnerabilities.
- Generate test reports and issue lists for the development team.
