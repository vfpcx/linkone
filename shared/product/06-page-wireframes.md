## 0. 文档说明

ASCII 线框 + 关键交互注释，作为前端开发的视觉骨架。**不包含**视觉风格、配色、组件库选型（归前端 Agent）。

图例：

```
┌──┐ 容器 / 卡片        [btn] 按钮         < xxx > 输入框
( ) 单选  [x] 多选     ⌄ 下拉             🎤 语音入口
📷 拍照入口            🗺️ 地图             📍 坐标 / 距离
🟢🟡🔴 状态色          ※ 说明文字（非 UI）
```
